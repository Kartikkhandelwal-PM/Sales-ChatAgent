How to View and Download Return Register Reports in ExpressTDS

The Return Register in ExpressTDS is a comprehensive module that records and tracks all TDS returns filed by a Deductor across various financial periods in a single-page view. It serves as a detailed log, offering clear insights into the Deductor’s filing activities, providing a streamlined and organized summary of all TDS returns for efficient tracking and compliance.

Steps to View and Download Return Register Reports:

Step 1: Access the ExpressTDS portal by entering your username and password. Once logged in, the Deductor Master page will appear, displaying a list of all deductors.

Step 2: In the Deductor Master page, click on the "Return Register" button.

      

Step 3: A new page will open, showing all the returns filed by the Deductor. This page is divided into two sections:

Return Wise: Displays the returns organized by return type.

Deductor Wise: Shows the returns organized by Deductor.
Step 4: To download a specific Return Receipt/Acknowledgement, click the PDF icon. This will download the selected return receipt.

      

Step 5: To view returns by Deductor, click the "Deductor Wise" tab.

      

Note: After using the "Sync Now" button, the data should always reflect the most up-to-date information.

      

By following these steps, you can easily view and download Return Register reports in ExpressTDS, allowing you to keep track of all TDS filing activities.
