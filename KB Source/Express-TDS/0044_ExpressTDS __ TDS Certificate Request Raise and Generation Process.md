TDS Certificate Request Raise and Generation Process

Step 1: Select Company/Deductor 

Begin by selecting the appropriate company or deductor from the available list.

Step 2: Access TDS Certificate Option 

Navigate to and select the TDS Certificate option.

      

Step 3: Specify Form Year and Quarter

Select the Relevant form, year and quarter, then proceed by clicking Submit Request.

      

Step 4: Provide Request Details

Fill in the required details to submit the request.
Alternatively, you can import a TXT file for the selected quarter to autofill the details.

Click Proceed to continue.

      

Step 5: Await ZIP File from TRACES

Once the request is successfully submitted, wait for the ZIP file to be generated by TRACES.
To check the request status, use the Status Check option.

If the request is not available:
Raise a New Request: Navigate to TRACES and select "Raise a New Request".
                  
Update the Status: Sync ExpressTDS with TRACES to refresh the status of previously submitted requests.
                  
Enter the CAPTCHA and click Submit to verify the update.

Step 6: Generate PDF Certificate

Once the ZIP file is available, choose one of the following options:

Generate Unsigned PDF

Generate DSC-Signed PDF (requires a valid digital signature token)

      

Step 7: Download the Generated PDFs

Once the PDFs are successfully generated, use the Download option.

A single ZIP file containing all PDFs will be available for download and can be saved for record-keeping.

      

      

            

By following these steps, you can efficiently raise, track and download TDS Certificates using ExpressTDS, ensuring compliance and seamless management of TDS records.
