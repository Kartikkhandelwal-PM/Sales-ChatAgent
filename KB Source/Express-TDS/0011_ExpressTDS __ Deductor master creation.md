Deductor Master Creation Guide for ExpressTDS

This guide provides instructions on how to create a Deductor Master in ExpressTDS using three methods:
Using ITD Credentials: Add deductors by entering ITD login details.
Bulk Excel Import: Create multiple deductors simultaneously using an Excel file.
Importing TXT and TDS/Conso Files: Add multiple deductors by uploading TXT or TDS/Conso files.

      
Using ITD Credentials

Step 1: Login to www.expresstds.com and click on "Add Deductor".

      

Step 2: Enter your Income Tax Portal credentials and Traces Credentials with Passwords.

      
      

Step 3: Click "Proceed" to automatically add the company to the Deductor Master.

Bulk Excel Import
Step 1: Select "Add Deductor" on the Deductor Master page.

Step 2: Click the "Browse Excel" option to download the provided Excel template.

      

      

Step 3: Open the template and fill in the following details:

Deductor Name

Deductor Username (TAN)

Income Tax Password

      

Step 4: Once filled, drag and drop the file into the system or click to select it for upload.

      

Step 5: The system will automatically begin creating deductor records based on the uploaded data.

      

Importing TXT and TDS/Conso Files
Step 1: Click "Add Deductor" on the Deductor Master page.

Step 2: Select the "Browse Folder" option to locate the TEXT or TDS/Conso file.

      

Step 3: Navigate to the file location and click "Import" to upload the data.

      

Step 4: Confirm the import by clicking "Okay".

      

Step 5: For each client, enter the corresponding password, then click "Create Client" to finalize.

Step 6: Click "Save" to complete the process.

Note : By using the "Auto Scan" button, the application can automatically search for and locate any available TXT and TDS/Conso files on your system. Once identified, it will import these files into the application, streamlining the data import process.

      

By following these steps, you can efficiently create a Deductor Master using any of the three methods in ExpressTDS.
