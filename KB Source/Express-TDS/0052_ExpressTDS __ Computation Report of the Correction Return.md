Purpose of the Computation Check

The computation check is essential for ensuring the accuracy and completeness of the prepared return. This step is crucial for:
Verifying Reporting: Facilitates a detailed review of the data reported in the return.
Error Prevention: Identifies potential errors or discrepancies, helping to avoid warnings or issues during the File Validation Utility (FVU) generation and filing process.
Ensuring Compliance: Confirms that the return meets all regulatory requirements.
These checks are vital for ensuring a smooth filing experience and maintaining compliance with tax regulations.

Step-by-Step Process for Computation Check

Step 1: Navigate to the Deductor Master
Select the relevant company or deductor from the list.
Go to the Return Summary and choose the created return.
Step 2: View the Created Return
Click View for the selected return.

      
Step 3: Import the Consolidated File
After importing the consolidated file, proceed to the next step.
Step 4: Start the Correction Process
Click on the Start Correction option.

      
Step 5: Select the Computation Option
Locate and click on the Computation option to access the computation check along with a complete summary.

      

      
Here, you can verify the computation details and make necessary corrections.
