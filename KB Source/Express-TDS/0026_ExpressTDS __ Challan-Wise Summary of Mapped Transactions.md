Procedure to View Mapped Challan Transactions in ExpressTDS
Begin by logging into the ExpressTDS platform using your credentials.
Accessing Mapped Challan Transactions

Option 1: Using Challans Overview

Select Deductor: Choose the appropriate deductor from the available list.

Go to Dashboard: Navigate to the dashboard and select the "Challan Overview" option.
      
View Mapped Transactions: Locate the transaction number associated with the challan and click on it to view the detailed mappings.
      

Option 2: Using Computation and Health

Access the Return Register: Click on the "Return Register" and select the relevant return view option.
      
Navigate to Computation and Health: Go to the "Computation and Health" section and select "Challan-Wise Summary of Mapped Transactions".
      
View Deductee Mappings: Identify and click on the transaction number to review the deductee's mapped to the respective challan.
      

Additional Note:
These steps ensure the accurate mapping of challans and associated data. Verify that all returns are prepared within the software and that relevant data is available for validation.
