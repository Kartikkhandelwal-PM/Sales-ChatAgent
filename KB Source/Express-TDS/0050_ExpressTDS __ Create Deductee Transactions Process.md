Steps to Create Deductee Transactions in a TDS Return

Step 1: Navigate to the Deductor Master and select the relevant company or deductor. Go to the Return Summary and select the created return.

      

Step 2: Click the "View" option for the created return.

      

Step 3: Add Deductee Transactions → You can add deductee transactions in two ways: Manual Addition and BULK Import

Manual Addition:

Click on the "Add" icon.

      
A deductee creation table will appear.

      
Enter the transaction details, including the deductee name and PAN.

Click "Save" to create the transaction.

Bulk Import:

Click on the "Import Return" option and download the Excel template.

      

      
Fill in the deductee details in the Excel template and save it.

      
Import the saved Excel file back into the software using the "Import Return" option.

      
Select either the "Append" or "Overwrite" option accordingly, then submit.

      

By following these steps, you can create single or bulk deductee transactions for TDS Returns.
