Steps to Import Transactions via Text File (Challan/Deductee Entry)

Follow the steps below to import transaction records into the system using a TXT file:

Step 1: Navigate to the Deductor Master

Go to the Deductor Master section.

Select the relevant company or deductor.

Open the Return Summary and select the return you want to work on.

Step 2: View the Created Return

Click on the "View" option for the selected return.

      

Step 3: Initiate the Import Process

Click on the "Import Return" button.

      
Use the Drag and Drop feature or click to browse and locate the file.

Step 4: Upload and Import the TXT File

Select the required TXT file for the return.

Choose the Overwrite option to replace existing data, if necessary.

      
Click on Import to complete the process.

      

By following these steps, the transaction data from the TXT file will be successfully imported into the system.
