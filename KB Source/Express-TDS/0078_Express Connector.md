Express Connector – Installation & Troubleshooting Guide

This document provides step-by-step instructions to install, verify, and troubleshoot the Express Connector to ensure uninterrupted application functionality.

Overview: If you receive an error message related to the Express Connector, it typically indicates one of the following:

The connector is not installed.
The installed version is outdated.
The connector service is not running.
Browser security settings are restricting required permissions.

      
      

Please follow the instructions below to resolve the issue.

Step 1: Download the Express Connector

Log in to the application.
Download the latest available version of the Express Connector.

      
            

Step 2: Install the Express Connector

Open the Downloads folder on your system.
Locate the downloaded installer file.
Double-click the file to initiate the installation process.
Upon successful installation, the following confirmation message will appear: 
“The installation has completed successfully! Express Connector is ready to use.”

Click OK to close the installer.
Step 3: Verify Connector Status via Task Manager

After installation, please confirm that the connector service is running:
Open Task Manager (Ctrl + Shift + Esc).
Go to the Details tab.
Verify that ExpressConnector2.exe is listed and running.

If the process is not visible:
Restart your system.

Reopen Task Manager and verify again.
If the process still does not appear, reinstall the connector.

Step 4: Enable Required Browser Permissions (Google Chrome / Microsoft Edge)

When accessing the application via Google Chrome or Microsoft Edge, certain browser permissions must be enabled.

Recommended Method: Using Site Settings
Open the application in your browser.
Click the “View Site Information” icon located next to the address bar.
Locate and enable the following permissions:

Local Network Access

Apps on Device

Refresh the page or restart the browser to apply the changes.

Important: If these permissions are disabled, the application may not communicate properly with the Express Connector.

Conclusion
To ensure proper functionality of the application:

Install the latest version of the Express Connector.
Confirm that the connector service is actively running.
Enable the required browser permissions.

If the issue persists after completing all steps outlined above, please contact the Support Team and provide relevant screenshots or error messages for further assistance.
