---

The Express RECO module is a
web-based reconciliation solution designed to facilitate accurate and efficient
comparison of Books data with 26AS/AIS (Annual Information Statement).

The module is accessible via:

app.kdksoftware.com
expresstds.com

It provides a structured workflow
for client creation, party mapping, transaction mapping, and reconciliation,
ensuring data accuracy, transparency, and compliance.

Key
Features
1.
26AS/AIS Reconciliation – Books vs. 26AS/AIS
Express RECO enables seamless
reconciliation between Books and 26AS/AIS data, allowing users to identify
discrepancies, validate transactions, and ensure accurate reporting.

2.
Client Creation
The system offers a simplified and
secure process for creating and managing clients:
User-friendly interface for quick client onboarding
Minimal steps for efficient client setup
In-built validations to ensure accurate data entry
Secure framework to protect sensitive client
     information
3.
Party Mapping

Party Mapping ensures accurate
alignment of entities between Books and 26AS/AIS:
Enables precise mapping of parties across both data sources
Reduces reconciliation errors through structured
     mapping
Provides detailed reports for audit and compliance
     purposes 

Allows smooth navigation and review of mapped records
4.
Transaction Mapping
Transaction Mapping provides
detailed control and flexibility in reconciliation:
Categorization of transactions based on:
Section
Transaction Date
Tax Value
Status
Detailed transaction-level insights for better analysis
Identification of missing or unmatched transactions
Configurable tolerance limits as per business
     requirements
Improved accuracy through customizable mapping criteria
5.
Reconciliation & Analysis
The module provides comprehensive
insights to support decision-making:
Consolidated view of reconciled and unreconciled
     transactions
Advanced filtering based on status and matching
     parameters
Risk identification between Books and 26AS/AIS data
Enables follow-up actions with Deductors for TDS
     compliance
Intuitive interface for easy data review and navigation
6.
Navigation & Usability
Express RECO is designed for ease of
use and quick implementation:
Direct access to the Reconciliation Summary
     Dashboard
Smooth and consistent navigation across all modules
Optimized interface for faster processing and minimal
     learning curve
