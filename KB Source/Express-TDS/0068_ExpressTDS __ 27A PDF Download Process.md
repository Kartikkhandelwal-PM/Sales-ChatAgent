Follow these steps to download Form 27A -

Step 1: Navigate to Deductor Master
Log in to the system and navigate to the Deductor Master section.
Select the relevant company or deductor.
Go to the Return Summary and choose the created return.

      

Step 2: View the Created Return
Click the View option for the selected return.

      
Step 3: Validate and Upload
After thoroughly reviewing the details, click on Validate and Upload.

      
Choose one of the following options based on the return type:Generate FVU (IT Portal): For returns uploaded directly to the Income Tax Portal.
Generate FVU (OLTAS): For returns uploaded via the OLTAS system.

      

Step 4: Download the FVU File
A ZIP file will be generated. Download and extract the ZIP file.

      
Step 5: Access Form 27A PDF
After extracting the ZIP file, locate the folder and view the Form 27A PDF.

      

For Correction Returns
The same process applies to correction returns. After generating and downloading the ZIP file for the correction return, extract the file to access the Form 27A PDF.
In the extracted folder, you can view the 27A PDF Form.

This streamlined process ensures you can easily access Form 27A for both original and correction returns.
