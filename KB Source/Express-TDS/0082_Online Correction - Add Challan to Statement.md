# Online Correction - Add Challan to Statement

This feature allows the deductor to add a challan to an already filed statement. The Add Challan functionality is not available through offline correction. If the statement is not processed by TRACES, please file C1 (Personal Information) through the Conso file. Once processed, you can proceed with online correction on TRACES.

Steps for Online Correction:

Step 1: Login to the TRACES Website

Log in to the ExpressTDS portal using your username and password. The Deductor Master page will open, displaying a list of all deductors.

From the Deductor Master, click on the desired deductor to access their details page.

In the selected deductor's page, review the relevant information.

Locate the TRACES icon in the deductor's page.

      
Click on the TRACES icon to be redirected to the TRACES portal login page.

      

Step 2: Navigate to Request for Correction under the Defaults menu.

      

      
Step 3: Enter the relevant details: Financial Year, Quarter, Form Type, and select Correction Category as "Online". Click on the Submit Request button.

      

      
The request will be available under Track Correction Request.

Once the request status changes to Available, click on Available/In Progress to proceed with the correction.

      

Step 4: Validate KYC
Note: The Digital Signature-supported KYC validation screen will appear only if the Digital Signature is registered.

      

      

Normal KYC

      

      

      

      

Step 5: After successful KYC validation, select the Type of Correction Category from the dropdown as Add Challan to Statement.

      

      

Click on Add Challan (Minor Head Code-200/400)

      

      

      

      

      

      

User can remove added challan through ‘Remove Challan’ option.

      

Note: Only newly added challans can be removed from the statement. Challans that were already present in the statement cannot be removed.

      

      

Add Challan (Minor Head Code-100)

      

      

      

      

      

Step 6: Make the required corrections in the selected file and click on Submit for Processing to submit the correction. (Only available to the Admin User)

      

      

      

      

      

Step 7: A 15-digit token number will be generated and sent to the registered email ID.

      

      

      

      

By following this process, you can successfully add a challan to the statement via online correction.
