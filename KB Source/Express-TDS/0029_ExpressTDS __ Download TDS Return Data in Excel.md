Steps to Download TDS Return Data in Excel Format via ExpressTDS

You can download TDS return data for both regular and correction returns in Excel format. Ensure transactions have been created in the return before proceeding. Here’s how:

Download TDS Data for Regular Returns

Process 1: Through Return Summary

Navigate to Deductor Master → After logging in, go to the Deductor Master section to access TDS returns and company details.
Select the Company → Choose the relevant company for which you want to download TDS data.
Download in Excel Format → Look for an Excel icon or Download option within the TDS return section. Click the icon to download the TDS return in Excel format.
      

Process 2: Through Deductee Transaction Page

View Created Return → Go to the Return Summary section. Click on View button.
      

Access Deductee Transaction Summary → Within the return details page, locate the Deductee Transaction Summary section to view all deductee transactions.
Download Return Data in Excel → In the Deductee Transaction Summary section, click Export or Download to save the data in Excel format.
      

Export Error Records (if any) → If errors exist, find the Error Record or Error Summary option. Click on it to export any error data in Excel.
      

Download TDS Data for Correction Returns

Navigate to the Correction Return → Go to the Return Summary section and click View next to the Correction Return you want to download.
      

Import Consolidated File (if not yet imported) → If the required consolidated file hasn’t been imported, do so at this step.
Export Data from Quick Links Table → If you need to update or correct the return, go to the Quick Links table. Click Import Return from Excel if you are importing data for corrections.
      

Export the Corrected Return to Excel → After making necessary corrections, click Export Data to download the corrected return in Excel format.
      

By following these steps, you can efficiently download TDS return data for both regular and correction returns in ExpressTDS, ensuring all details are available in an editable Excel format.
