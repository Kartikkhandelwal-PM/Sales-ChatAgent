TDS Return PDF Download in ExpressTDS

Follow these steps to download the TDS return in PDF format:

Step 1: Navigate to Deductor Master

Log in to ExpressTDS and go to the Deductor Master section to access TDS returns and company details.

Step 2: Select the Company

Choose the relevant company from the available list for which you want to download the TDS return data.

Step 3: Select the Return to Download

From the available returns, select the TDS return you need to download in PDF format.

      

Step 4: Download in PDF Format

Look for the PDF icon or Download option within the Return Summary section.

Click on the PDF icon to download the TDS return in PDF format.

      

Important Note: Make sure that the TDS return data is filed through ExpressTDS or that the data has been imported into the return. Only then will you be able to download the return in PDF format.

By following these steps, you can easily download your TDS return in PDF format for your records.
