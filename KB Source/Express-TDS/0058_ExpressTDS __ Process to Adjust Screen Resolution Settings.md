Issue Description:

In certain scenarios, users may experience difficulties where specific options, such as Validate and Upload, are not displayed. This issue is typically caused by improper screen resolution settings.

Resolution Steps:

1. Adjust System Screen Resolution:

Right-click on the desktop and select Display Settings.
      
Navigate to the Display Resolution section.

Set the resolution to 100% or the recommended setting.
      
After making the adjustment, restore the resolution to its default 100% setting.

Verify that all options, are now visible.

2. Modify Browser Display Settings (If Necessary):

If the issue persists, adjust the screen resolution settings within the specific browser where the cloud software is being accessed.

Navigate to the Settings menu of the browser and modify the resolution accordingly.
      
By following these steps, users should be able to resolve visibility issues and restore full functionality.
