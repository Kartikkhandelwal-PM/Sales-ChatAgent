# ExpressTDS || TDS Filed Return Acknowledgment Receipt Download

**Section:** Express TDS

**Category:** Express-TDS

**Source:** https://support.kdksoftware.com/portal/en/kb/articles/tds-filed-return-acknowledgment-download

**Tags:** tds filed return acknowledgment download, tds receipt download, filed return acknowledgment download, ack receipt download, receipt download, expresstds, tds filed return, acknowledgment download, tds receipt, acknowledgment receipt download, acknowledgment receipt

---

To download the TDS filed return acknowledgment on the Express TDS portal, you can choose one of two options:

Option 1: Download Acknowledgment via Return Register

Step 1: Login to Express TDS: Sign in to your Express TDS account.

Step 2: Go to Return Register: At the top right, select Return Register.

      

Step 3: Select the Return: Choose the return for which you want to download the acknowledgment.

      

Step 4: Download Acknowledgment: Click the PDF icon next to the selected return to download the acknowledgment receipt.

      

Note: If you don't see the data, use the Sync option to refresh.

      

Option 2: Download Acknowledgment via Return Summary

Step 1: Login to Express TDS: Sign in to your Express TDS account.

Step 2: Select Deductor/Company: Choose the deductor or company for which the return was filed.

      

Step 3: Go to Return Summary:

For returns filed in Express TDS: In the Created Return Table, find the return and click the PDF icon to download the acknowledgment.

      
For returns filed outside Express TDS: Go to Other Returns Filed Outside Express TDS and locate the return. Click the PDF icon to download.

      

Note: Use the Sync option if the data isn't visible to update it.

By following these steps, you should be able to quickly download your Income Tax acknowledgment receipt after filing.
