Deleting a Challan in Regular Return

Step 1: Access the Deductor Master

After logging in ExpressTDS, navigate to the Deductor Master Dashboard.
Select the relevant company or deductor from the list provided.

      

Step 2: View Created Returns

In the Returns Created section, locate the list of previously generated returns.

Click the View button next to the return for which you want to delete a challan.

      

Step 3: Delete the Challan

Within the selected return, go to the All Challans section.

      
Select the challan you wish to delete.

Click the Delete icon.

      
Confirm your action by clicking Delete again to finalize the process.

      

By following these steps, you can efficiently delete a challan in a Regular Return.
