Minimum Java Requirement:

Java 8 or above → To check the installed version, navigate to Control Panel → Programs and Features.

Troubleshooting Java Errors

Step 1. Verify Java and Express Connector Installation

Open Control Panel → Programs and Features

Confirm that Java and the Express Connector are installed.

Step 2. Automatic Java Installation

Run the Express Connector as Administrator to allow automatic Java installation.

Restart the system after installation.

Step 3. Manual Java Installation (If the Issue Persists)

Verify the installed Java version (Control Panel → Programs and Features)

If required, download and install Java 8 Update 251 (32-bit version)

Restart the system and proceed with TDS certificate generation

By following these steps, Java-related issues should be resolved, ensuring smooth processing of TDS certificate generation.
