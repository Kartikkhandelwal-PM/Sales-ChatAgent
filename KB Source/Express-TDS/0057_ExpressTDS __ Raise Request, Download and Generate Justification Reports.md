Steps to Raise Request, Download, and Generate Justification Reports in Express TDS

In Express TDS, you can manage Justification Reports by raising requests, downloading, and generating them using the following methods:

Raising a Request for Justification Report

You can raise a request for Justification in two ways:

Process 1: Request from Deductor Dashboard

Open the Deductor Dashboard.

To raise a Justification request, refer to this guide for detailed steps - Click Here.
Process 2: Request from Notices & Orders

Go to Notices & Orders from the main screen.

When you synchronize the software with TRACES, any notices available will show in this section with a Raise Request option.

For detailed guidance, refer to this article - Click Here.

Downloading the Justification Report

Once the request is raised, follow these steps to download it:

Wait for Processing: Once the request is raised, the status will show as "Submitted", Wait for 2 to 3 hours for the request to process.

      
Update Status from TRACES: After the waiting period, go back to the Request Raised to TRACES screen and click Update Status from TRACES. The status should change to "Available".

      
Download the File: Once available, click on the download option to retrieve the ZIP file. This file can then be accessed from the Notices & Orders section.

Generating the Justification Report

There are two ways to generate the Justification Report:

Process 1: Generate via Excel
Go to Notices & Orders and locate the updated request in the Request Raised to TRACES section.

Click on the Excel option to download the file.

            
Open the downloaded Excel file to view the department’s summary, organized sheet-wise.

      
Process 2: Generate via Software (View)
In Notices & Orders, locate the updated request and click on the View option next to Excel.

      
The Justification Report will open directly within the software for easy viewing.

      

By following these steps, you can efficiently raise, download, and generate Justification Reports in Express TDS.
