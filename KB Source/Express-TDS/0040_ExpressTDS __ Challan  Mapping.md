Summary of the Three Methods for Mapping Challans

Process 1: Manual Mapping

Open the Deductee Transactions and click the action icon.

Select Mapped Challan to individually assign a challan to each deductee transaction.

      

Process 2: Excel Export and Import

Export the Excel template and ensure challan details are filled in.

In Deductee Transactions, map each challan using the dropdown cell.

Import the completed Excel file back into the software to upload the mapped challan details.

      

Process 3: Bulk TDS Mapping

Select multiple Deductee Transactions at once.

Drag and drop them onto the selected challan shown on the interface.

      

      

This structured approach simplifies the challan mapping process and accommodates various user preferences.
