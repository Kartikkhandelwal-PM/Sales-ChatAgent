Here’s a step-by-step guide on how to view and prepare the HSN summary for the GSTR-9 return in ExpressGST software:

Steps to View and Prepare HSN Summary in ExpressGST

Step 1: Access GSTR-9 Status

Log in to ExpressGST: Open the ExpressGST software and log in using your credentials.

Navigate to the Return Summary Section: On the dashboard, locate and access the Return Summary section.

Click on GSTR-9 Status: Select the GSTR-9 Status option to view the current status of your annual return filing.

      
Step 2: Access the HSN Summary Section

Click on the "Others" Option: In the GSTR-9 section, find the 6th option labeled as Others.

      

1. HSN-Wise Summary of Outward Supplies

Find the 17th Option: Look for the 17th option labeled as HSN-Wise Summary of Outward Supplies.

      
Auto-Populate HSN Summary from GSTR-1:

Use the Populate HSN Summary from GSTR-1  feature to auto-populate the HSN details.

                        
If errors occur during the auto-population process, manually select the tables with errors and correct them.

                        

                        
Manual Entry: If you prefer to manually enter data -

Use the Export option to download an Excel sheet.

                        
Fill in the HSN details in the Excel sheet.

                        
Import the completed sheet back into ExpressGST by following the import steps.

                        

NOTE: The Excel Import feature will append data to the existing entries.
If the details are already present in the table, delete the existing entries entirely before importing the prepared Excel sheet.

                        

                        

2. HSN-Wise Summary of Inward Supplies

Find the 18th Option: This is for the HSN-Wise Summary of Inward Supplies.

      
Process for Inward Supplies:

Similar to outward supplies, export the details, fill in the Excel sheet, and import the data back into ExpressGST.

                        

                        
                        

                        

                        

This process ensures that the HSN summary for both outward and inward supplies is accurately prepared for your GSTR-9 return.
