# ExpressGST || Import of GSTR-1A Books Data

**Section:** Express GST

**Category:** Express-GST

**Source:** https://support.kdksoftware.com/portal/en/kb/articles/import-of-gstr-1a-books-data

**Tags:** gstr-1a book, expressgst, gstr-1a, import of gstr-1a books data, gstn blank template, gst, import books data, gstr 1a, expressgst - import of gstr-1a books data, gstr-1a books, gstr-1a books data, import process, gstr

---

This document provides a detailed
procedure for importing GSTR-1A Books Data into the ExpressGST application.
Adhering to this process ensures accurate data upload and facilitates seamless
return preparation and filing.

Step
1: Login and Client Selection

Log in to the ExpressGST application
and select the relevant client for whom the GSTR-1A Books Data is to be
imported.

Upon selection, the system will automatically redirect to the Return Summary page.

Step
2: Access Import Module

On the Return Summary page, all
applicable GSTR return types will be displayed.

Navigate to the left-hand panel and click on Import Books Data to
initiate the import wizard.

      

Note: The GSTR-1A return import feature is enabled only after the successful filing of GSTR-1 and remains available until the due date of GSTR-3B.

Step
3: Selection of Return Period and Data File

Navigate to: Sales → GSTR-1A

Select the applicable Return Period (Month and
     Year) for which the data is to be imported (e.g., March 2026).
Download the required Excel template, as applicable: 

GSTN Blank Template – Available under the GSTN Template option
Consolidated Excel Template (KDK Template) – Available under the CA Plan for simplified and
      structured return preparation
Note: The KDK Template is a consolidated
worksheet within the GSTR-1A Excel format that captures section-wise summary
details of outward supplies, including:
Taxable Value
CGST 

SGST 

IGST 

Cess 

The data is systematically derived
from detailed transaction categories such as B2B, B2C, Exports, and
Credit/Debit Notes, and consolidated into a single sheet to facilitate
efficient data import.

Click on the Browse button corresponding to the
     selected return period. 

Select the prepared Excel file from your system and
     click Open to upload the file. 

Step
4: Initiate Import Process

Upon successful file selection,
click on the Import button to initiate the data import process.

Step
5: Review Import Status

The system will display an import
status message indicating whether the data has been successfully imported or if
any discrepancies/errors are identified.

Step
6: Confirmation and Completion

Once the import process is
successfully completed:

Click OK to confirm the action. 

The imported GSTR-1A Books Data will be available for
     review, validation, and further processing within the application. 

Important
Notes

Ensure that the Excel file strictly adheres to the
     prescribed template format. 

Verify that the correct return period is selected prior
     to import. 

In case of import errors, review the file structure and
     data format before re-attempting the upload. 

Following the above standardized
procedure ensures accurate and efficient import of GSTR-1A Books Data into
ExpressGST, thereby enabling smooth return preparation and compliance with GST
regulations.
