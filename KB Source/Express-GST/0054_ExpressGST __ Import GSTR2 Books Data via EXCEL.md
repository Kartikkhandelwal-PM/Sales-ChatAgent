Process for Importing GSTR-2 Book Data via Excel into Express GST

Note: We Support Govt. and all Accounting Software Templates.

Step 1: Select the Client
Log in to Express GST and navigate to the Client List.
Select the client for whom you wish to import the GSTR-2 book data.
This will take you to the Return Summary page for the selected client.
Step 2: Access the Import Books Data Feature
On the Return Summary page, locate the Import Books Data button on the left-hand side of the screen.
Click the Import Books Data button to open the import wizard.

      

In the wizard, select GSTR-2 and then click on the Excel option.
      
Step 3: Download the Template (Optional)
If you don’t have the template for the GSTR-2 book data, click the Blank Template option to download the required Excel file.

      
Fill out the template with the necessary purchase details.

      
Step 4: Import the Prepared Excel File
Once your Excel sheet is prepared, return to the wizard and use the Browse button to select and upload the GSTR-2 book data Excel file.

      
Step 5: Verify the Imported Data
After the file is imported, navigate to the Books vs 2A/2B Annual section.

      
In this section, verify and reconcile the imported book data against the portal's purchase data, focusing on an invoice-wise comparison to ensure accuracy.

      

By following these steps, you can successfully import and reconcile your GSTR-2 Books data with the portal's purchase data in the Express GST system. This ensures that all relevant purchase information is captured accurately, streamlining your GST filing and reconciliation process.
