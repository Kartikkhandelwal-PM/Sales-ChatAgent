# ExpressGST || Return Register

**Section:** Express GST

**Category:** Express-GST

**Source:** https://support.kdksoftware.com/portal/en/kb/articles/returnregister

---

The Return Register in ExpressGST is a comprehensive dashboard that provides both a summary and detailed view of all GST returns filed by a business (client). It serves as a central hub where users can track, manage, and monitor the status of various GST returns, ensuring full compliance with GST filing requirements.

Client-Wise feature in the Return Register of ExpressGST allows users, such as tax consultants or businesses managing multiple GSTINs, to view and manage GST returns on a per-client basis. This is particularly useful for organizations handling GST compliance for multiple clients or branches under different GSTINs.

Month-Wise feature in the Return Register of ExpressGST organizes and displays GST returns based on specific tax periods, typically monthly or quarterly, depending on the taxpayer's filing frequency. This feature helps users track and manage GST return filings across different months, ensuring adherence to deadlines and filing schedules.

Steps to Access the Return Register :

Step 1: Access the ExpressGST portal, once logged in, the Client Master page will be displayed, listing all clients.

      

Step 2: To view GST returns, click on the "Return Register" option.
To view returns client-wise, click on the "Client-Wise" option.

      
To view returns month-wise, click on the "Month-Wise" option.

      

Step 3: To download a client-wise summary in Excel, click on the Excel button.

      

By following these steps, you will successfully access and navigate the Return Register in ExpressGST.
