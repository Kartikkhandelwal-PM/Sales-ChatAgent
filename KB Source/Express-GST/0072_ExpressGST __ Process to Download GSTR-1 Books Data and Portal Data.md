# ExpressGST || Process to Download GSTR-1 Books Data and Portal Data

**Category:** Express-GST

To download GSTR-1 Books Data and Portal Data in Excel from ExpressGST, follow these straightforward steps for each type of data.

Step 1: Select the Client in Client Master

Navigate to the Client Master and select the relevant client.
Step 2: Select the Month

From the Return Summary section, pick the desired month for which data is required.

Step 3: Check the Status

Click on the appropriate status option: Filled, Due, or Overdue depending on the data shown.

      

To Download Books Data

Select Data as per Books Option.

Click on the Books Data Download icon to export the data in Excel format.

      

To Download Portal Data

Select Data as per Portal Option.

Click on the Portal Data Download icon to export the Portal data in Excel format.

      

By following these steps, you can efficiently download both Books Data and Portal Data for GSTR-1 from ExpressGST in an organized Excel file.
