The revamped GSTR-1 module in ExpressGST is designed to provide a streamlined and structured workflow for return preparation, validation, and filing. It facilitates efficient data management through improved data entry interfaces, enhanced validation controls, and seamless integration with the GST portal.

Navigation to GSTR-1 (New Format)

To access the updated GSTR-1 interface:

Click on “Switch to New”

Select the applicable Return Period (Month and Year)

Click on “Prepare/Due” to proceed

Data Classification

The GSTR-1 data is organized into the following categories:

Original Entries – Transactions reported for the first time

Amendments – Modifications to previously filed data

Stage 1: Prepare Books (Data Entry Stage)

This stage allows users to input and manage return data prior to validation and filing.

Manual Data Entry

All relevant GSTIN-based GSTR-1 tables are available for direct data entry.

To add new data:

Navigate to the required table

Click on the “+” (Add) button

      

A structured data entry screen will open for capturing the required transaction details.

      

Multiple Line-Item Entry

For documents involving multiple line items:
Use the “+” option available on the right panel to add additional rows within the same document

      
This enables comprehensive entry of invoice-level details in a single record.

Editing Existing Data

Navigate to the relevant table containing the entry to be modified.

Select the specific record you intend to update.

Edit the required fields as applicable.

Verify the updated information for accuracy and completeness.

Press Enter to save the changes.

Best Practices

Ensure the correct return period is selected before initiating data entry or edits

Validate all entries to avoid errors during reconciliation and filing stages

Maintain consistency in invoice and GSTIN details to prevent mismatches

By following the above process, users can efficiently manage GSTR-1 data within ExpressGST. The structured workflow ensures accuracy, minimizes errors, and supports compliant and timely return filing.
