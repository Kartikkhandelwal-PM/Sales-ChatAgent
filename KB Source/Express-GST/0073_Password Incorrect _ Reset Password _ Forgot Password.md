# Password Incorrect / Reset Password / Forgot Password

**Section:** Process

**Category:** Express-GST

**Source:** https://support.kdksoftware.com/portal/en/kb/articles/password-incorrect-reset-password-forgot-password

**Tags:** reset password, password incorrect, password is incorrect try again, expressgst, password is incorrect, password reset, try again, expresstds, forgot password, expressitr

---

If you see the error message "Password is incorrect, try again" while logging into applications like ExpressGST, ExpressITR, ExpressTDS or any similar application, it typically means that the password entered is incorrect.

      

Solution →
Double-check the password you entered. If it was entered incorrectly, simply re-enter the correct password and try logging in again.

If you continue to see the error message "Password is incorrect, try again," you will need to reset your password.

To Reset Your Password, Please follow these steps:

Step 1: Click on the "Forgot Password?" option.

      

Step 2: Enter your Registered mobile number. You will receive an OTP on this number for verification.

      

Step 3: Enter the OTP.

      
      
Step 4: Create a new password, confirm it and click on Update Password.

      

You can now log in with your updated password.
