This document outlines the standard procedure to import GSTR-1 Books data into the ExpressGST application. The process supports both JSON and Excel-based imports, enabling efficient data integration for return preparation and filing.

Step 1: Login and Client Selection

Log in to the application and select the relevant client for whom the GSTR-1 Books data is to be imported. 

Upon selection, the system will navigate to the Return Summary page.

Step 2: Access the Import Books Data Module

From the Return Summary page, locate the Import Books Data option available on the left-hand panel and click on it to initiate the import process.

      

Step 3: Select Return Period and Data File

Navigate to Sales → GSTR-1 and select the applicable Return Period (Month and Year) (e.g., March 2026).

      

A. Import via JSON File

Click on Browse Files

Select the required JSON file

Click on Import to proceed

      

B. Import via Excel File

Download and prepare the data using any of the following templates:

GSTN Blank Template (available under GSTN Template option)

Consolidated Excel Template (KDK Template) (available under CA Plan)

Amazon Template – For importing Amazon MTR data

Flipkart Template – For importing Flipkart sales data

      

Note on KDK Template:

The KDK Template is a consolidated worksheet within the GSTR-1 Excel format that captures section-wise summaries of outward supplies, including Taxable Value, CGST, SGST, IGST, and Cess. The data is derived from detailed transaction categories such as B2B, B2C, Exports, and Credit/Debit Notes and presented in a structured format for seamless import.
Click on the Browse option corresponding to the selected return period, select the prepared Excel file from your system, and click Open to upload.

B2C Data Import Configuration
Append to existing B2CS data → The B2CS records from the import file will be appended to the existing dataset. Previously available records will remain unchanged.

Overwrite existing B2CS data → The existing B2CS data will be completely replaced with the records from the import file.

Not applicable → The system will exclude B2CS data from the import process under this option.

      
​

Step 4: Initiate Import Process

After successfully uploading the file, click on the Import button to start the data import process.

Step 5: Verify Import Status

The system will display a confirmation message indicating the status of the import process, including whether the data has been imported successfully or if any discrepancies are identified.

Step 6: Confirmation and Completion

Upon successful import, click OK to confirm. The imported GSTR-1 Books data will be available for review, validation, and further processing within the application.

Adhering to the above procedure ensures accurate and efficient import of GSTR-1 Books data into ExpressGST, thereby facilitating a smooth and compliant return preparation and filing process.
