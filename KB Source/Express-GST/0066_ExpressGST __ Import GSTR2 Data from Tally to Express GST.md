ExpressGST | Importing GSTR-2 Data from Tally

The following steps and prerequisites are essential for accurately importing GSTR-2 data from Tally into ExpressGST. Meeting these criteria ensures precise mapping and successful data import.

Prerequisites for Importing GSTR-2 Data from Tally to ExpressGST
Matching GSTIN

Ensure the GSTIN (Goods and Services Tax Identification Number) in both Tally and ExpressGST is identical.
Any discrepancy in GSTIN will prevent the system from accurately mapping or importing the data.

Matching Port Code
The Port Code in both Tally and ExpressGST must match to correctly identify import/export locations.
Mismatches may result in unsuccessful data imports.

By aligning both the GSTIN and Port Code, ExpressGST can seamlessly map and import the GSTR-2 data from Tally, ensuring an efficient and error-free process.

How to Import GSTR-2 Data from Tally into ExpressGST
Step 1: Select the Client
Log in to ExpressGST and select the client whose GSTR-2 data needs to be imported. This will take you to the Return Summary page.
Step 2: Access Import Books Data
On the Return Summary page, locate the Import Books Data button on the left and click it.

      
Step 3: Configure Import for GSTR-2 Data
Click Import Books Data and select the GSTR-2 option.

      
Click Configure with Tally.

      
A wizard will open, providing two options for importing GSTR-2 data from Tally:
My PC → Select this option if Tally is installed and running on the same system as the ExpressGST application, allowing direct data import without a network connection.

Client PC → Choose this option if Tally is on a separate computer. The client will receive a syncing link via email to facilitate the data sync with ExpressGST.

Importing GSTR-2 Data on My PC

Step 1: Select “My PC” → Choosing "My PC" indicates that Tally is installed and active on the same system as ExpressGST.

      

Step 2: Initiate the Tally Import Process → Click My PC to proceed. This will display companies in Tally with mapped GSTIN and Port Numbers to ExpressGST.

      

Step 3: Map the Selected Company → Choose the desired company and click Map. Tally will confirm the mapping with a message.

      

Step 4: Sync GSTR-2 Data → If Auto Sync is disabled, manually initiate the sync by clicking Sync Now to import the GSTR-2 data from Tally into ExpressGST.

      

Step 5: Complete the Sync Process → Once syncing is complete, a window will display the status of the imported GSTR-2 data. Click Close to exit the Tally Sync Status window.

      

Step 6: Comparison View → Go to Books vs 2A/2B Annual and choose Supplier-wise or Invoice-wise Reconciliation for a detailed comparison.

Note: Use filters like Matched, Partially Matched, In Books Only, and In 2A/2B Only for clarity.

Importing GSTR-2 Data on Client PC

Note : Client PC refers to a different computer where Tally is installed, which is separate from the machine running the ExpressGST application. When you choose the Client PC option, it allows you to remotely import GSTR-2 data from that other computer. The client (the person using Tally on the remote computer) will receive a syncing link via email to facilitate the data transfer between their system and the ExpressGST application on your system.

Step 1: Select Client PC → Click Import Books Data, then choose Client PC.

      

Step 2: Enter Client Details → Fill in the client’s name, email, and phone number, and then click Send Request to Client.

      

Step 3: Syncing Process for Client → The client will receive an email with a syncing link. Instruct the client to open the email and click Take Action to start syncing.

      

Step 4: Install Express Connector → For the initial setup, install the Express Connector by clicking Take Action once more after installation.

​

By following these steps, GSTR-2 data can be efficiently imported from Tally into ExpressGST, supporting streamlined and accurate data management.
