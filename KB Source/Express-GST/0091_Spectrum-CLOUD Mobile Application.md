# Spectrum-CLOUD Mobile Application

**Section:** Process

**Category:** Express-GST

**Source:** https://support.kdksoftware.com/portal/en/kb/articles/spectrum-cloud-mobile-application

**Tags:** tds, spectrum cloud mobile application, expressgst, spectrum-cloud, spectrum cloud web application, itr, ledgers, mobile application, expresstds, spectrum cloud, gst, spectrum-cloud mobile app, expressitr, notices, returns, spectrum-cloud mobile application, spectrum cloud mobile app, mobile app

---

The Spectrum CLOUD Mobile Application enables users to securely access Spectrum Cloud data directly from their mobile devices. The application provides seamless access to GST, TDS, and ITR-related information, allowing users to monitor critical compliance activities anytime and anywhere through a secure, efficient, and user-friendly interface.

Key Features & Enhancements

1. Mobile – ITR Notices

Users can now access Income Tax Return (ITR) notices directly through the mobile application.

Features

View notice summaries

Access detailed notice information

Financial Year-wise notice mapping for easy reference
2. Mobile – TDS Notices

The application now supports mobile access to TDS-related notices, enabling users to manage and monitor notices more efficiently.

Features

View detailed notice information

Track notice status in real time

Identify the applicable section under which the notice has been issued

Integrated workflow support with existing TDS processes
3. Mobile Utilities

GSTIN Search → A quick and efficient utility for verifying GSTIN details.
Available Information

Legal Name

Registration Status

State Jurisdiction
TDS Rate Finder → Simplifies the process of identifying applicable TDS rates.
Benefits

Section-wise TDS rate identification

Helps minimize manual errors during deduction processing
Find HSN Code → Enables users to search and identify the appropriate HSN code through keyword-based lookup functionality.

IT New Session → Allows users to initiate a new Income Tax session directly from the mobile application with improved performance and seamless session initialization.

The application is designed to provide fast, secure, and convenient access to essential compliance data through a simplified mobile interface.

Accessing the Application

Downloading the Application →  To download the application:

Open the Google Play Store or Apple App Store.

Search for KDK Software.

Download and install the application.
Sign-In Process → Existing Spectrum Cloud Web Application users can log in using their registered mobile number.
Steps to Sign In

Enter the registered mobile number.

An OTP (One-Time Password) will be sent to the registered mobile number.

Enter the OTP to complete authentication and securely access the application.
Client Master → The Client Master section displays all companies configured within the Spectrum Cloud Web Application.
Companies are automatically categorized under the following modules:

GST

TDS

ITR
Accessing Company Information → To access company-related information:

Select the required company from the list.

Choose the appropriate module to view the relevant details.
GST Module → The GST Module provides comprehensive access to GST-related information for the selected company.
Available Sections

Notices

Ledgers

Returns
Notices → The Notices section enables users to view GST notices and orders received for the selected company.
Viewing Notice Details

Click the View (Eye) icon corresponding to the notice.

The application will display detailed information related to the selected notice or order.

Available Notice Information

ARN Number

Application / Case Creation Date

Notice Sub-Type

Status

Order Description

Order Number

Issued Date

Due Date

Demand Amount

Reply Details

This functionality helps users efficiently monitor notices and track compliance activities.

Ledgers → The Ledgers section provides access to various GST ledgers associated with the selected company.
Users can:

View ledger details directly within the application

Download ledger reports in Excel format for detailed analysis and record-keeping

Available Ledgers

Cash Ledger → Displays tax head-wise balances along with the total available balance in the Cash Ledger.
RCM Liability / ITC Statement → Provides details of Reverse Charge Mechanism (RCM) liabilities and Input Tax Credit (ITC) balances.
Negative Liability Statement → Displays details related to negative liability adjustments along with tax head-wise information.
Electronic Credit Reversal → Provides information regarding credit reversals recorded in the electronic credit ledger.
Credit Ledger → Displays Input Tax Credit balances with tax head-wise breakup.
Non-Return Liability Ledger → Shows liabilities arising due to non-filing of returns, including detailed tax head-wise balances.

Ledger Options → Each ledger provides the following options:

View (Eye) Icon – Access detailed ledger transactions

Date Range Filter – Review transactions for a selected period
Returns

The Returns section allows users to monitor the filing status of GST returns for the selected company.

Return Categories

Pending Returns

Overdue Returns

Filed Returns
Financial Year Selection → Users can select the applicable Financial Year (FY) to view return details for the respective period.

Return Information Views

The application provides two convenient viewing modes for return information.

Month View → The Month View enables users to:

Select a specific month

View return details for the selected month

Download or share return information
Return View → The Return View enables users to:

Select a specific return type

View filing status month-wise

Check return filing dates

Available GST Returns → The following GST returns can be viewed through the mobile application:

GSTR-1

GSTR-1A

GSTR-3B

GSTR-9

GSTR-9C

The KDK Software Mobile Application is available for download on both the Google Play Store and the Apple App Store, enabling users to access Spectrum CLOUD services seamlessly across Android and iOS platforms.
