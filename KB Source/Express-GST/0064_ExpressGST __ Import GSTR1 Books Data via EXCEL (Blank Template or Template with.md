What is GSTR-1 PAN Level ?

GSTR-1 PAN Level refers to a consolidated report that provides detailed information about outward supplies (sales) made by a taxpayer, organized at the PAN (Permanent Account Number) level. This report consolidates data from multiple GSTINs (Goods and Services Tax Identification Numbers) that are registered under the same PAN, streamlining GST filing for businesses with multiple registrations.

Process for Import GSTR-1 Books Data via EXCEL (Blank Template or Template with PAN Level)

To import GSTR-1 books data via an Excel sheet into Express GST, follow these steps:

Step 1 : Choose the client for whom you want to import GSTR-1 book data by clicking on the client’s name. This will take you to the Return Summary page.

Step 2 : On the Return Summary page, you will see a list of all types of GSTR filings. Locate the Import Books Data button on the left-hand side.

      

Step 3 : Select or Download Template:

Click on the Import Books Data button to open a wizard.

      
You can either select your GSTR-1 book data Excel file or Download a Blank template or Template with PAN-level if needed.

      

      

NOTE: We Support Govt. and All Accounting Software Templates.

Step 4 : Upload GSTR-1 Book Data File

Under Sales GSTR-1, choose the relevant month and year (e.g., October 2024) for which you want to import data.

Click the Browse button next to the selected month, and locate the GSTR-1 book data Excel file on your computer.

      
Click Open to upload the file to the Express GST portal.

      

Step 5 : Once the file is uploaded, click the Import button to start the data import process.

      

Step 6 : A window will appear displaying the import status of the GSTR-1 book data.

      

Step 7 : Once the import is completed, click OK to finalize the process. The successfully imported GSTR-1 book data will now be visible in the Express GST portal.

By following these steps, you can efficiently import GSTR-1 book data into Express GST and ensure accurate filing of your GSTR-1 returns.
