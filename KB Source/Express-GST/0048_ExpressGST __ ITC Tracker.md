ITC Tracker – Overview and User Guide

Overview

The ITC Tracker module is designed to provide
structured monitoring, reconciliation, and action management of Input Tax
Credit (ITC) with direct alignment to GSTR-3B Table 4.

It enables organizations and tax professionals to track the
entire lifecycle of ITC and ensure that actions taken on invoice data are
accurately reflected in GSTR-3B reporting.
The module follows a structured workflow:

Reconcile → Track → Take Action → Reflect in GSTR-3B Table 4 Summary

Through this process, users can manage ITC across monthly,
quarterly, and annual periods while maintaining proper compliance controls.

The dashboard provides consolidated visibility of:

ITC
     available as per GSTR-2B

Temporary
     reversals

Permanent
     reversals

Reclaimable
     ITC balances

ITC
     already claimed

Pending
     or unresolved actions

The ITC Tracker also provides multiple analytical views for
efficient data analysis and action management, including:

Records are categorized across different document types such
as Invoices, Debit Notes, Credit Notes, Reverse Charge Mechanism (RCM), and
Imports, allowing users to monitor ITC eligibility and reconciliation
status more effectively.
The module also tracks upward and downward amendments
separately, enabling users to identify additional credits or reductions
arising from supplier amendments and corrections.
The Invoice Action Center mirrors the GSTR-2B
dataset, allowing users to take structured actions such as:
Claim
     ITC
Temporary

Permanent

Reclaim

Each action automatically updates the Table 4 summary
view, ensuring alignment between the tracker and GSTR-3B filing data.
Additional features such as Reversal and Reclaim
tracking, Excel-based bulk actions, and automated validation alerts help
users identify discrepancies and maintain compliance.

      

ITC Tracker – Process Flow

The ITC Tracker module enables systematic reconciliation,
action tracking, and reporting of ITC data with alignment to GSTR-3B
Table 4.
 

Step 1: Select the Company

Select
     the company for which ITC data needs to be reviewed and managed.
Step 2: Access the ITC Tracker
Navigate to Reconciliation Table → ITC Tracker.

      

This will open the ITC Tracker dashboard where
reconciliation data and actions can be managed.
 

Step 3: Import Book Data

Use the Import Book Data option to import accounting
records for reconciliation with GSTR-2B data.

      

After importing the data, records will be categorized based
on reconciliation results, such as:
Matched
     records
Partially
     matched records
Books-only
     records
2B-only
     records
POS
     mismatch cases
ITC
     eligibility conflicts
These classifications assist users in identifying
discrepancies and taking appropriate actions.

Default System Behaviour

When
     the ITC Tracker is opened, the system automatically selects the due
     month of the currently pending GSTR-3B return.

By
     default, records with “Pending / No Action” status are displayed.

Users
     can enable the “Show All” option to view all available records.

Table 4 – Action Summary

After actions are taken within the ITC Tracker, the results
are reflected in the Table 4 – Action Summary.

      
      

This summary helps users review the final ITC position
and prepare or update Table 4 of GSTR-3B accordingly.
