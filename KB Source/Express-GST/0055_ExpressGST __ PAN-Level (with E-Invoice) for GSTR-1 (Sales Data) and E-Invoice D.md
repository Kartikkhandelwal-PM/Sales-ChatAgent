PAN-Level (with E-Invoice) for GSTR-1(Sales Data) and E-Invoice Data Import

Introducing a Unified Template

With the new PAN-Level with E-Invoice template, users can now seamlessly manage GSTR-1 (Sales Data) and E-Invoice Data Import using a single format. This eliminates the need for multiple templates, ensuring:

Simplified Workflow: A single template for both GSTR-1 and e-invoice imports.

Enhanced Accuracy: Reduces manual input errors and ensures alignment between Sales Data (GSTR-1) and e-invoice records.

Steps for Data Import

Step 1: Import Books Data
Navigate to the Import Books Data option.

      

Step 2: Download the Blank Template

On the summary details page, select the PAN Level (with E-Invoice) option.

      
Download the blank template sheet.

      

Step 3: Fill the Template

Enter the required data in the downloaded template as per the specified format.

Step 4: Import the Filled Template

Once the template is prepared, upload it into the software.

Select the relevant Month for which the return is being filed.

      
Click Import to upload the file.

Step 5: Verify and Confirm

After importing, the software will display the details from the uploaded sheet.

      
Review the data for accuracy and click OK to confirm.

Step 6: Check the Return Status

Navigate to the Return Summary section.

Check the status under Pending Return Due/Overdue Status.

      
Ensure all details are accurate before proceeding with the filing.

      

E-Invoice Generation Process

Step 1: Generate E-Invoice

After importing the template, initiate e-invoice generation from the Profile section of the software.

Click on E-Invoice and proceed.

      

Step 2: Select GSTN

Choose the relevant GSTN (Goods and Services Tax Identification Number) for which the e-invoice is to be generated.

      

Step 3: Review E-Invoice Status

The software will display a Status Summary of the imported data, including e-invoice details.

      
      
Step 4: Generate and Email the E-Invoice

Once the e-invoice details are verified, click Generate & Email.

      
The system will generate the e-invoice and send it to the registered email address.

This streamlined process ensures efficient data import, e-invoice generation, and return filing using the ExpressGST platform.
