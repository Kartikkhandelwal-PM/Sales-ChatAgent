Guide to Creating an Auditor Master

Step 1: Log in to the software and navigate to the Auditor Master section.

      

Step 2: Click the Add (+) button to add a new auditor.

      

Step 3: Enter the required details for the auditor, including:

Auditor Name

Proprietorship/Partnership/LLP Name

Membership Number

ICAI Registration Number

Proprietorship/Partnership/LLP PAN
Address etc.

      

Once all the details are entered, click the Save button to store the information.

Step 4: To edit an auditor's details, use the Edit button.

      

Step 5: To activate or deactivate an auditor, use the Switch button under the Active column.
