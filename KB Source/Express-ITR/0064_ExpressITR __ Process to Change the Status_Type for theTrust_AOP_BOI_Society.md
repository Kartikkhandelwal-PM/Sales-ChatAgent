Stepwise Process to Update Status/Type

Step 1: Log in to the Software

Access the software using your credentials.

Navigate to the Client Master section.

Step 2: Select the Record

Locate the specific record for the Trust, AOP, BOI or Society.

Click the Edit button in the Action column.

      

Step 3: Update the Status/Type

Open the Personal Information section.

Select the desired Status/Type from the drop-down menu.

      

Step 4: Save Changes

Click the Save button to confirm and apply the updated information.

Step 5: Verify the Update

Review the updated record to ensure the changes have been successfully applied.

By completing this process, you have successfully updated the Status/Type in the record for the Trust, AOP, BOI or Society.
