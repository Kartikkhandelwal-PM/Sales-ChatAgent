Importing Capital Gain Data Using Broker Template

The Broker Template feature in Express ITR enables the structured and accurate import of Capital Gain data. Transactions imported through this functionality are automatically classified under the appropriate income heads, including Capital Gains and Business or Profession Income, wherever applicable.

This feature ensures standardized data formatting, minimizes manual intervention, reduces the risk of errors, and enhances overall efficiency in return preparation.

Step 1: Access Client Profile

Log in to the Express ITR application.

Select the required client from the Client List.

Navigate to the ITR Summary section.

Click on Return Flow to proceed.

Step 2: Navigate to Capital Gain Section

Click on the Capital Gain tab.

The Capital Gain data entry screen will be displayed.

Step 3: Initiate Import Process

Click on the Import CG Statement option.

The import interface will open for data upload.

Step 4: Import Capital Gain Data

Users can import data using any of the following methods:

Method A: Import via Broker Credentials

Enter the required details based on the respective broker:

Zerodha / Groww – User ID and Password

Upstox – Registered Mobile Number

Angel One – Registered Mobile Number or Client ID

Kotak Securities – Upload broker file using drag-and-drop

Upon successful authentication, the system will automatically fetch and import the Capital Gain data.

Method B: CAMS / KFINTECH Mailback Statement

CAMS and KFINTECH provide a facility to generate a consolidated account statement (Mailback Statement) for mutual fund investments.

Steps to Import:

Request the consolidated statement using the registered email ID.

Download the password-protected Excel file received via email.

Unprotect the file using the password defined at the time of request.

Important:  The password required is the Excel sheet unprotect password set during the request process. It is not the broker login password.

Upload the unprotected file using the Browse File option in the application.

Method C: Upload Broker Template File (Recommended)

Select and upload the predefined Broker Template Excel file.

Key Advantages:

Ensures uniform data structure

Enables accurate classification of transactions

Facilitates automatic population of Capital Gain records

The system will process and populate the data automatically upon successful upload.

Step 5: Validate and Complete Import

Click on the Import button after selecting the file.

Review the confirmation message and click OK.

Output
Upon successful import, the data will be reflected under:

Capital Gains

Business or Profession Income (where applicable)

This confirms that all broker transactions have been successfully imported and appropriately classified.

Conclusion

The Broker Template feature provides a reliable and efficient method for importing Capital Gain data into Express ITR. It improves data accuracy, reduces manual effort, and streamlines the overall return preparation process.
