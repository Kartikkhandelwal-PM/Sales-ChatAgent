Revised Return Preparation Process in Express ITR

To prepare a revised return in Express ITR, follow these steps:

Step 1: Access the Revise Return Option
Open the client’s profile in the Express ITR application.

Navigate to the ITR Summary section and select the "Revised" option to initiate the process.

      

Step 2: Edit Original Data
In Revise return mode, you can access data from the original return.
Click on the Edit button or go to each income head to make the necessary adjustments.

      

Step 3: Review and Prepare for e-Filing
Once edits are complete, click the E-filing button.

      
Select Review and E-filing to ensure all details are accurate before submission.

      

Step 4: Submit the Revised Return
When ready, click on "Proceed for e-Filing" to submit the revised return to the Income Tax Department (ITD).

      

By following these steps, you can efficiently prepare and submit a revised return in Express ITR.
