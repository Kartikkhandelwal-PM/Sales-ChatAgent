# ExpressITR || Importing Data from the Previous Assessment Year

**Section:** Data Feeding

**Category:** Express-ITR

**Source:** https://support.kdksoftware.com/portal/en/kb/articles/importing-data-from-the-previous-assessment-year

**Tags:** import the previous year’s data, import from previous year, spectrum cloud itr, import previous year data, importing data from the previous assessment year, expressitr, client profile, data import, import previous year’s data

---

This article provides detailed instructions on how to import data from the previous Assessment Year (AY) into the current year’s Income Tax Return within ExpressITR. This feature helps ensure accuracy, continuity, and faster preparation of returns.

Steps to Import Previous Year Data

Step 1: Select the Client and Assessment Year

Open application and navigate to the required Client Profile.

Choose the Assessment Year for which you want to import the previous year’s data.

      

Step 2: Initiate the Data Import

After selecting the Assessment Year, click on the Pending button.

      

System Processing

Once the Pending button is clicked, ExpressITR automatically fetches the available data from the previous Assessment Year.

The imported details are populated under the respective Income Heads in the current year's ITR.

      
      
      

Importing data from the previous Assessment Year in ExpressITR is a simple and efficient process that minimizes manual effort and ensures consistency in financial reporting. By following the steps above, users can seamlessly carry forward relevant information and speed up the ITR preparation workflow.
