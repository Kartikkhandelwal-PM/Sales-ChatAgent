Steps to Import Capital Gain Data Using Excel
Step 1: Access the Client Profile
Open the client’s profile in the Express ITR application.
Navigate to the ITR Summary section within the profile.
Click the button under Return Flow to open the relevant ITR options.

      
Step 2: Select Capital Gain Section
Click on the Capital Gain heading to open the capital gain data entry area for this client.

      
Step 3: Download the Excel Template
Select Import CG Data and Download the provided Excel template, which is formatted for capital gain data entry.

      

      
Step 4: Enter Data in the Excel Template
Open the downloaded template and fill in the client’s capital gain data in the designated fields.

      
Ensure that all information is accurate and matches the required format.

Step 5: Import the Excel File
In the software, click Browse to select the completed Excel file from your system.

      
After selecting the file, click Import and confirm by clicking Okay.

      

      

With the above process, the capital gain data will be successfully imported into the software.
