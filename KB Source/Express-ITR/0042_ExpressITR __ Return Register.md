The Return Register in Express ITR is a centralized module that provides a detailed overview of all Income Tax Returns (ITRs) filed for clients. It displays essential information such as Filing Status, Filing Date, ITR Form Type, and also offers options to download the Form PDF and ITR Receipt PDF.

Steps to Access and Use the Return Register

Step 1: Log in to Express ITR

After logging into the ExpressITR module, the Client Master page will open, displaying all active clients associated with your account.
Step 2: Open the Return Register
Navigate to the left-hand menu and click on Return Register to access all return-related information for your clients.

      
Step 3: Select the Assessment Year
Choose the appropriate Assessment Year (AY) from the dropdown menu.
 The system will then display the filing status of all clients for the selected year in a consolidated view.
      

Step 4: View Filing Status and Download PDFs
Within the Return Register, you can perform the following actions:

View Filing Status: Check the latest filing status for all clients at a glance.

Download Form PDF: Download the ITR Form in PDF format.

      
Download ITR Receipt PDF: Download the ITR-V/acknowledgement receipt.

      

Note: Click the Return Status button to sync and fetch the most recent status from the Income Tax Department, ensuring accurate and updated information.

      

Step 5: Verify & Discard Options for Unverified Returns

To manage returns where the filing status is Filed – Unverified, follow the steps below:

Quick Access from Client Master

The system displays the total number of Unverified Returns directly on the Client Master screen.
Click ACT NOW to proceed with verification or discard actions instantly.

      
      

Standard Procedure via Return Register

Filter Unverified Returns

Open the Return Register and apply the filter: Filing Status = Filed – Unverified.

      
      

To Verify - Unverified Returns

Click the E-Verify button.

A confirmation dialog will appear. Review the message carefully and click Continue.

You will be redirected to Income Tax Department (ITD) Portal.

Select the preferred verification mode (Aadhaar OTP, DSC, or other available methods) and complete the verification.

      
      

To Discard - Unverified Returns
Click the Discard button.

Confirm the action by clicking Continue.

You will be redirected to the Income Tax Department (ITD) Portal.
A confirmation prompt will appear. Read the instructions carefully and click Accept and Continue to complete the discard process.

Using these options, users can efficiently e-verify or discard - Unverified returns with minimal effort.

The Return Register in Express ITR offers a streamlined and efficient way to monitor, verify, and manage all client return filings from a single interface. By following the above steps, users can quickly track filing statuses, download essential documents, and handle actions such as discarding e-verified returns with ease. This ensures better workflow management and improved accuracy in return-related processes.
