Here's a structured guide for preparing Regular Return or Belated Return using Express ITR:

Prepare Regular Return

Step 1: Access the Return Flow

Open the client profile in Express ITR.

Sync Data:

Click the Sync button to import data from 26AS or AIS (TDS, TCS, Tax Paid, or Income details).

Data will be automatically populated into the software.

      
Use the Auto-Fill button under each Head of Income to update the relevant fields.

Step 2: Enter Other Income Information

Manually input any additional income details that are not reported in the AIS.

      

Step 3: Verify Tax Computation

After entering all income data, review the Tax Computation to verify the accuracy of the fed data and the calculated taxes.

      

Step 4: Compare Tax Regimes

Review the Old vs. New Tax Regime comparison.

      
Select the tax regime that benefits the client the most.

      

      

Step 5: E-Filing Preparation

Verify all income and tax computation details.

If accurate, proceed by clicking the E-Filing button.

      

Step 6: Error Rectification

Click on the Review and E-Filing button.

      
Address errors flagged by the Schema Validation system:

Click on the error link to be redirected to the incomplete or incorrect data fields.

Update the information as required.

Step 7: Final Steps

Once all errors are cleared:

Download the JSON file using the Download JSON button.

Alternatively, click Proceed for E-Filing to file the return on the portal.

      

Prepare Belated Return

Follow the same process as above for filing a belated return.

Note: The software automatically identifies the return as Belated if filed after the due date.

Note: Always double-check all entries and computations to ensure the return is accurate and compliant with current tax laws.
