Importing a 3CD JSON File Process

To import a 3CD JSON file in Express ITR, follow these steps:

Step 1: Open Audit Summary

Access the client profile in Express ITR.

In the Audit Summary section, click on Import JSON to proceed.

      

Step 2: Import the JSON File

Click on the Import button and then select Continue to begin importing the 3CD JSON data.

      

Step 3: Confirmation of Import

Once the process completes, the 3CD data will be successfully imported into the software.

      

Following these steps ensures the 3CD JSON file is properly imported.
