# ExpressITR || JSON File Import Process

**Section:** Process

**Category:** Express-ITR

**Source:** https://support.kdksoftware.com/portal/en/kb/articles/json-file-import-process

**Tags:** filed json, spectrum-cloud, unfiled return, spectrum cloud, expressitr, importing json for a filed return, json file import, unfiled json, json import process, filed return, spectrum cloud itr, importing json for an unfiled return, import process, json import

---

This article explains the step-by-step procedure to import a JSON file into ExpressITR, applicable for both filed and unfiled returns.

ExpressITR allows users to seamlessly import JSON files generated from the Income Tax Department portal. The imported data automatically populates the relevant sections of the return, improving accuracy and reducing manual entry. This guide covers:
Importing JSON when the return is already filed
Importing JSON when the return is not yet filed

Importing JSON for a Filed Return

Step 1: Access the Import JSON Feature

Open the relevant Client Profile in ExpressITR.

Select the appropriate Assessment Year.
Navigate to the ITR Summary section.
Under Import JSON, click the Import button.

Click on the JSON Import button to proceed.

Step 2: Confirm the Import
Click Continue to confirm and initiate the JSON import process.

      
      

Importing JSON for an Unfiled Return (Unfilled JSON)

Step 1: Access the Import JSON Feature

Open the relevant Client Profile in ExpressITR.

Select the appropriate Assessment Year.
Navigate to the ITR Summary section.

Click the Import button under Import JSON.

Step 2: Select the JSON File

Click on the Import JSON button.
Browse and select the JSON file you wish to import.

The system will successfully import and populate the JSON data into the client’s ITR workspace.

By following the steps outlined above, you can easily import JSON data into ExpressITR, ensuring smooth and error-free preparation of both filed and unfiled returns.
