Download AIS/26AS in PDF Format

To download the AIS and 26AS reports in PDF format from Express ITR, follow these steps:

Step 1: Access the ITR Summary

Open the client’s profile in the Express ITR application.

Within the profile, navigate to the ITR Summary section to proceed.

      

Note:

The ExpressConnector file is required for data synchronization and all online activities within your application.

If a message appears regarding the ExpressConnector (see screenshot below), please download and install the ExpressConnector Setup to continue.

For a detailed installation guide for the ExpressConnector, refer to the link: ExpressConnector Installation Guide.

      

Step 2: Download AIS PDF

In the ITR Summary, locate the AIS option.

Click on the PDF icon below to AIS to initiate the download.

      
You will be redirected to the Income Tax Portal, where the AIS PDF file will automatically download and open.
Step 3: Download 26AS PDF

In the ITR Summary, locate the 26AS option.

Click on the PDF icon below to 26AS to start the download.

      
You will be redirected to the Income Tax Portal, where the 26AS PDF file will automatically download and open.
By following these steps, you can efficiently download the AIS and 26AS PDF files for your client.
