# ExpressITR || Form 16 Import Process

---

The Form 16 Import functionality in ExpressITR allows users to directly import salary and TDS details from the Form 16 file into the Income Tax Return (ITR) preparation module. This feature helps reduce manual data entry, improves accuracy, and simplifies the return preparation process.
This article explains the complete process for importing Form 16 data into the software.

Steps to Import Form 16 in ExpressITR

Step 1: Select the Client

Open the application and navigate to the Client Master section.

Select the relevant client for whom the Form 16 data is to be imported.

After selecting the client, the system will redirect to the client dashboard.

Step 2: Open the Return Flow

Click on the Return Flow option available for the selected client.

Select the applicable Assessment Year and return workflow.

This section contains all heads and schedules required for return preparation.

Step 3: Navigate to the Salary Head

Under the Return Flow, open the Salary head.

Click on the Import Form 16 option.

      
The Form 16 Import window will open on the screen.

Step 4: Select the Form 16 File

Click on the Browse or Select File option.

Navigate to the folder or drive where the Form 16 file is stored.

Select the required Form 16 file for import.

Ensure that the selected file is the correct Form 16 document related to the selected client and assessment year.

Step 5: Import the Form 16 Data

After selecting the file, verify the file path and details.

Click on the Import button to start the import process.

The software will validate and process the selected Form 16 file.

Step 6: Verify Imported Data

Upon successful import, the salary and TDS details from Form 16 will be automatically populated in the respective sections of the return.

Review the imported data carefully before proceeding further with return preparation and filing.

Important Points

Ensure that the correct client and assessment year are selected before importing the file.

The Form 16 file should be valid, accessible, and free from corruption.

Verify all imported values before final submission of the return.

In case of import failure or validation issues, recheck the selected file and try importing again.
