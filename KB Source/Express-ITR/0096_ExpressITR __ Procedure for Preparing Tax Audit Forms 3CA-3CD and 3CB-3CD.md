Procedure for Preparing Tax Audit Forms 3CA-3CD and 3CB-3CD

Step 1: Open the client and select the Prepare button under the Audit Summary section.

      

Step 2: Navigate to the Form 3CA-3CD/Form 3CB-3CD option. The appropriate form will be displayed based on the client's status.

      

Step 3: You will be directed to Form 3CA or Form 3CB, depending on the applicable scenario. Enter the auditor’s details, qualifications, comments and other relevant information.

      

Step 4: After completing the required fields, proceed to Form 3CD.

      

In Form 3CD, the clauses are divided into four parts as follows:

Clauses 4, 8–17

Clauses 18–27

Clauses 28–32

Clauses 33–44

You can also fill in the details using an Excel sheet. For the step-by-step process, refer to this guide: Import 3CD Data Using Excel

Step 5: Select the clause number for which details need to be provided. Use the options YES/NO/NA on the right side of the respective clause and input the required data.

      

If you need to download Form 3CD as a PDF with a different font size, follow this guide: Steps to Download Form 3CD as a PDF

Step 6: Upon completion of data entry, click on E-Filing to validate the return.

      

If validation errors are encountered, click the Error Message link. This will redirect you to the specific section requiring correction. Address the incomplete or incorrect information and revalidate the return.

      

Step 7: After successfully resolving all errors, download the JSON file and proceed to file the return.

      

If the return has already been filed using another application and you wish to import the data into our software, follow this guide: 3CD JSON File Import Process
